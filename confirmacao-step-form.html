PASSO 4: QUANDO PARAR A ENTREGA?

Função para pré-determinar o tempo total para seguir entregando um release. O padrão é 48h.
Caso queira, é possível diminuir o tempo padrão de entrega de acordo com suas necessidades. Ex: um disparo feito hoje (às 10h) para divulgar um evento no mesmo dia (às 18h). Se o usuário escolher a opção 12h, o sistema entregará os releases até 22h deste mesmo dia (10h + 12h). Depois das 22h, cessará as tentativas.
Vantagem: evitar que profissionais que estejam com caixa cheia, entre outros, recebam divulgações (releases, convites) desatualizadas.
